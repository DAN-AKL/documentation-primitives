## Names

`<Row>` components accept children so that the table can be used for tokens.

Use the `<Var>` component to format the tokens as code and give them the "copy to clipboard" functionality.

<Preview>
  <Story name="tokens">
    <SpacingScale>
      <Row value={4}>
        <Var>$spacing-01</Var>
      </Row>
      <Row value={8}>
        <Var>$spacing-02</Var>
      </Row>
      <Row value={12}>
        <Var>$spacing-03</Var>
      </Row>
      <Row value={20}>
        <Var>$spacing-04</Var>
      </Row>

        <Var>$spacing-05</Var>

        <Var>$spacing-06</Var>
