### "Name" column

The first column simply lists the data passed into the `Row` sub-components.

<Info>
  To format text as variables wrap it in a <code>Var</code> component. It uses the <code>code</code> tag internally and also adds the "copy to clipboard" functionality.
</Info>

  <Story name="tokens">

### Values

To make spacing values available for calculations these are passed in as numbers. The CSS unit options are <code>px</code> (default) and <code>rem</code>.

  <Story name="rems">
