# Design tokens

Design tokens are design decisions, represented as data.

## Introduction

Design tokens are the visual design atoms of the design system — specifically, they are named entities that store visual design attributes.

We use design tokens in place of hard-coded values — such as hex values for color or pixel values for spacing. We do this in order to maintain a scalable and consistent visual system for UI development.

<Warning prefix="Heads up">
  A new specification of design tokens will be released soon.
</Warning>

## Installation

Our design tokens are available as a standalone `npm` package.

<Info>We export tokens in CSS, SCSS and JavaScript formats.</Info>
