# Edit on GitHub

Opens specified files in the GitHub file editor.

---

<Preview>
  <Story name="default">
    <EditLink path="docs/primitives/EditLink.story.mdx" />
  </Story>
</Preview>

### Repository

The component expects the repository details added to the [Storybook configuration](https://storybook.js.org/docs/configurations/options-parameter/). The branch is set to `master` by default. Use this config to overwrite that if you link to files in other branches.

```js
// .storybook/preview.js
import { addParameters } from '@storybook/react';

addParameters({
  // other options
  repository: {
    baseUrl: 'https://github.com/<owner>/<repository>',
    branch: 'master',
  },
});
```

### Relative path to file

URLs are constructed using the repository details above and the relative path to the file supplied via the `path` prop. It should be relative to the root of your repository. For example, the path to _this_ file is `src/EditLink/EditLink.stories.mdx`.
